## en-US

There are `primary` button, `default` button, `dashed` button and `danger` button in antd.

````jsx
import { ComingSoon } from 'fui'

ReactDOM.render(
  <div>
    <ComingSoon
      title="维保选址"
      desc="马上要上线了，敬请期待！"
    />
  </div>,
  mountNode);
````
